INICI RÀPID (5 minuts) 🚀
══════════════════════════════════════════════════════════════════

1. Obre una terminal i navega al directori:
   cd techshop-teoarandapaez

2. Crea un entorn virtual:
   python3 -m venv venv

3. Activa l'entorn virtual:
   
   macOS/Linux:
   source venv/bin/activate
   
   Windows:
   venv\Scripts\activate

4. Instal·la les dependències:
   pip install -r requirements.txt

5. Inicialitza la base de dades (15 productes):
   python init_db.py

6. Executa l'aplicació:
   python app.py

7. Obre el navegador:
   http://localhost:5000

✅ LLEST! L'aplicació hauria d'estar funcionant!

PROBLEMES COMUNS I SOLUCIONS 🔧
══════════════════════════════════════════════════════════════════

❌ Error: "No module named 'flask'"
   ✅ Solució: pip install -r requirements.txt

❌ Error: "Port 5000 in use"
   ✅ Solució: Canvia el port a app.py (línia final)
              app.run(debug=True, port=5001)

❌ Error: "Database is locked"
   ✅ Solució: Tanca altres connexions a techshop.db

❌ La base de dades no es crea
   ✅ Solució: python init_db.py --reset
